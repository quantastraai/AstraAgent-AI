# AstraAgent Admin Dashboard

Separate admin dashboard for AstraAgent AI — SaaS-style UI for managing conversations, tickets, leads, and integrations.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Screens

- **Overview** — Stats cards, recent activity, weekly chart, recent conversations table
- **Conversations** — Intercom-style split view (list + chat window)
- **Tickets** — Ticket management table
- **Leads** — Leads table with source filters
- **Integrations** — Connection cards (WhatsApp, Zoho CRM, Email, Slack)
